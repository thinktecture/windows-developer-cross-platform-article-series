Welcome

Welcome to this cross-platform sample application. This sample shows various aspects of cross-platformed developed Angular applications.

Feature list

  • Angular - well, that's obvious, isn't it?
  • Angular CLI for dev, prod and hmr
  • Real responsive menu/sidebar/tabbar- heavily inspired by Scott Jensen
  • Cordova for mobile apps
  • Electron for desktop applications
  • Angular PWA support with Service Worker and Push Notifications
